# 경우의 수 — 학생 직접 유도 수업 설계 초안

**대상**: 고2 확률과 통계(2022 개정), 2학기 · 경우의 수
**모델**: 핵심 개념마다 도입 **5~10분** 학생 유도 → 이후 교과서 진도(개념·예제·문제·중단원·대단원 충실)
**관통 원리(단원 중심축)**: *"먼저 순서·기준을 정해 빠짐없이 세고, 같은 것으로 세어진 만큼 보정한다."*

> 평가 증거(활동보다 먼저 고정):
> **학생은 곱·합(또는 나눗셈 보정)을 쓴 *이유를 자기 말로 설명*하고, *처음 보는 실생활 상황에 같은 원리를 스스로 적용*한다.**

---

## PART 1. 5~10분 유도 루틴 템플릿 (반복 사용)

핵심 설계 원칙은 인터뷰에서 잡은 세 가지 위험을 그대로 막는다.

- "유도쇼"(교사가 유도하고 학생은 받아적기) 방지 → **학생이 먼저 틀리게 세고**, 그 틀림에서 보정을 발견.
- "모두 공존"(수준 편차) 대응 → **층위형 발문**(누구나 시작 가능, 빠른 학생은 일반화로).
- "실생활이 본질을 흐림" 방지 → 맥락은 **미끼로만**, 곧바로 수학 구조로 환원.

### 4비트 구조 (총 5~10분)

| 비트 | 시간 | 학생이 하는 것 | 교사의 핵심 발문 | 노리는 것 |
|---|---|---|---|---|
| ① 일단 세기 | 1분 | 작은 수의 실생활 상황을 **무작정 다 적어/세어 본다** | "일단 빠짐없이 적어볼까?" | 전원 진입(기초도 가능), 무작정 나열을 *허용* |
| ② 일부러 막히기 | 2분 | 수를 키운 같은 상황 → **누락·중복을 체감** | "이번엔 다 적기 귀찮지? 빠뜨리거나 겹치진 않아?" | 오개념(무작정 나열의 한계)을 **의도적으로 노출** |
| ③ 기준 세워 고치기 | 2~3분 | **순서·기준을 학생이 제안**하고 보정(곱/나눗셈) 발견 | "어떻게 하면 빠짐없이·겹치지 않게 셀까?" | 원리를 *학생이* 만들어 냄 |
| ④ 말하고 옮기기 | 1~2분 | **왜 그렇게 했는지 한 문장** + 처음 보는 상황 1개에 적용 | "왜 이렇게 셌는지 한 줄로. 그럼 이건?" | 평가 증거(설명+전이) **즉시 수집** |

### 층위형 발문(같은 비트, 3수준 동시 운영)

- **기초**: 작은 수 유지 + 표/격자 같은 구체 도구 제공, "다 적어보기" 끝까지 허용.
- **표준**: 수를 키워 "다 적기 힘들다 → 기준 세워 세기"로 유도.
- **심화(★)**: "n으로 일반화하면?", "다른 방법으로도 세어봐. 왜 답이 같지?"

### 빈 템플릿 (개념마다 채워 쓰기)

```
[개념]: ____________________
[관통 원리와의 연결 한 줄]: ____________________

① 일단 세기  — 실생활 미끼(작은 수): ____________________
② 일부러 막히기 — 수를 키운 버전: ____________________
③ 기준 세워 고치기 — 학생이 발견할 보정(곱/나누기): ____________________
④ 말하고 옮기기 — 설명 발문 / 전이 문항: ____________________

오개념 신호(이러면 안 됨): ____________________
exit ticket 2문항(설명 1 + 전이 1): ____________________
차등: 기초 ___ / 표준 ___ / 심화 ___
낮게 나오면 다음 차시에 바꿀 것: ____________________
```

### 단원 전체에서의 재사용 (관통 원리로 연결)

| 개념 | ③에서 학생이 발견할 보정 |
|---|---|
| 곱의 법칙 | '그리고'면 **곱**, '또는'이면 합 |
| 순열 nPr | 앞자리부터 줄여 곱하기(순서가 의미를 만든다) |
| 조합 nCr | 순열로 셈 → **같은 묶음이 r!번 중복** → **÷ r!** |
| 같은 것이 있는 순열 | 겹친 것끼리 중복된 만큼 **÷ (각 중복수!)** |
| 중복조합 | 칸막이·기호로 **다른 셈으로 환원** |

> 한 줄 요약: **거의 모든 개념이 ③에서 "왜 곱하나 / 왜 나누나(보정)"로 귀결**된다. 이게 단원을 하나로 묶는 축.

## PART 2. 곱의 법칙 — 템플릿 인스턴스 (완성 실연)

### 1) 학습 증거 문장 (관찰 가능한 학생 행동)
학생은 주어진 상황에서 **'그리고/또는'을 구분해 곱·합을 선택한 이유를 자기 말로 설명**하고, **처음 보는 실생활 상황에 곱의 법칙을 스스로 적용**한다.

### 2) 성공 기준
- 곱/합 중 옳게 선택한다.
- 선택 이유를 "공식이라서"가 아니라 **상황 구조('~하고 또 ~한다')로 설명**한다.
- 새 상황에서 같은 판단을 재현한다.

### 3) 간단 루브릭 (3수준)

| 수준 | 모습 |
|---|---|
| **우수** | 곱/합 정확 선택 + 구조로 이유 설명 + 새 상황 전이 성공 |
| **보통** | 선택은 맞지만 이유가 "공식/외워서" 수준, 전이는 일부만 |
| **미흡** | 곱·합 선택이 흔들림, 무작정 나열에 의존, 이유 못 댐 |

### 4) 오개념 신호 (비트·exit에서 관찰)
- '또는' 상황인데 **곱한다**(혹은 그 반대).
- 곱·합을 **무작위로 고르고 이유를 못 댄다**.
- 나열만 하고 "음료 하나당 디저트 매번 ○가지" 구조를 못 본다.

### 5) 최소 평가 장면 — 4비트 실연

**소재**: 카페 세트 = 음료 + 디저트

- **① 일단 세기 (1분)** — "음료 3종, 디저트 4종. 만들 수 있는 세트 다 적어봐." → 학생들 12개 나열.
- **② 일부러 막히기 (2분)** — "음료 5종, 디저트 6종이면? 30개 다 적을래?" → 귀찮음·누락 체감.
- **③ 기준 세워 고치기 (2~3분)** — "음료를 *하나 정하면* 디저트는 매번 몇 가지?"(6) → "음료 5개니까 5×6." **'그리고'=곱** 발견.
  - 곧바로 대비: "이번엔 음료 *또는* 디저트 중 **하나만** 고른다면?" → 5+6. **'또는'=합** 구분.
  - ★심화: "음료 a, 디저트 b면? 왜 a×b인지 한 문장으로."
- **④ 말하고 옮기기 (1~2분)** — "왜 곱했는지 한 줄." + 전이: "OX 문제 3개의 답안은 몇 가지?"(2×2×2) 또는 "상의 4·하의 3 코디 수?"

### 6) 수업 활동 흐름 (5~10분 후 교과서로)
유도 루틴(위) → 교과서 곱의 법칙 개념·예제 → 문제 → 합/곱 구분 추가 예제 → 중단원 문제로 누적. *유도에서 만든 "그리고=곱, 또는=합" 문장을 교과서 풀이 내내 판단 기준으로 재사용.*

### 7) exit ticket (3분, 증거 수집)
1. **(설명)** 다음 상황에서 곱했는지 더했는지 쓰고, *왜* 그런지 한 문장으로: "분식 메뉴 김밥 4종 *그리고* 음료 3종으로 만드는 세트 수." 
2. **(전이)** 처음 보는 상황 — "현관 비밀번호: 앞에 색깔 3개 중 1개 *그리고* 뒤에 숫자 2자리(00~99). 가짓수는?" → 3×100.

### 8) 차등 경로 (모두 공존 대응)
- **기초**: 음료/디저트를 **2차원 표(격자)**로 칠하며 "한 줄에 ○칸" 시각화, 작은 수 유지.
- **표준**: 수를 키워 곱으로 환원.
- **심화**: '그리고/또는'이 **섞인 복합 상황**, 격자·트리 두 표현이 같은 답을 주는 이유 설명.

### 9) 피드백 루프 (낮게 나오면 다음 차시에 바꿀 것)
exit 1번에서 **'또는→곱' 오류가 많으면**, 다음 차시 도입 1분을 "또는' 상황 먼저 제시 → 합으로 셈" 대비 활동으로 연다.

## 교사용 검토 체크리스트
- [ ] 평가 증거(설명+전이)가 활동보다 먼저 정해졌는가
- [ ] ③에서 원리를 **학생이** 발견하는가(교사 유도쇼 아님)
- [ ] 층위형 발문으로 기초·심화가 동시에 진입 가능한가
- [ ] 실생활 소재가 본질을 흐리지 않고 빠르게 수학 구조로 환원되는가
- [ ] exit 결과를 다음 차시 수정과 연결했는가
- [ ] 교과서 진도와 유도가 **같은 판단 문장**으로 이어지는가

## 교사 판단 필요(미확정)
- **실생활 소재**: 카페·분식·비밀번호는 예시일 뿐 — 학급 정서/관심사에 맞게 교사가 교체.
- **시간**: 반응 보고 5분↔10분 조절. 막힘(②)이 안 생기면 수를 더 키울 것.
- **모둠/기기**: 짝 토론·격자 활동 가능 여부는 교실 여건에 따라 교사가 결정.
